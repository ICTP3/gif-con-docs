# スタジオジフリ

## 概要
「スタジオジフリ」は、動画ファイルをブラウザ上でGIF画像に変換できるWebアプリケーションです。  
手元の環境に専用ソフトをインストールする必要がなく、簡単な操作でGIFを生成できます。

## 主な機能
- 動画ファイルをアップロードしてGIFに変換
- 変換したGIFファイルをダウンロード可能

## 工夫点
バックエンドの処理に Go言語 (Golang) を採用することで、  
変換処理を高速かつ安定的に実行できるようにしています。

## 使用技術
- フロントエンド：HTML / CSS / JavaScript
- バックエンド：Go (net/http)
- ツール：FFmpeg（動画 → GIF 変換）

## 動作環境
- Go 1.20 以上（推奨）
- FFmpeg がインストールされていること
